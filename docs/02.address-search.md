# Phase 3: 주소 검색 → 주변 CU 매장 → 행사상품 조회 기능 (2026-05-27)

## 작업 내용

주소/장소 검색 시 KakaoMap API와 연동하여 해당 위치 주변의 CU 편의점을 찾고, 선택한 CU 매장의 행사 상품(1+1, 2+1)을 조회할 수 있는 기능 구현.

---

## 데이터베이스 (MySQL)

### 기존 테이블 활용
`mart`와 `product` 테이블은 이미 존재하여 새로 생성하지 않음.

**mart** — 편의점 매장 정보
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | bigint (PK, AUTO_INCREMENT) | |
| name | varchar(50) | 편의점명 (CU) |
| address | varchar(200) | 주소 |
| latitude | double | 위도 |
| longitude | double | 경도 |

**product** — 행사 상품 정보
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | bigint (PK, AUTO_INCREMENT) | |
| mart_id | bigint (FK → mart.id) | CU 매장 FK |
| name | varchar(100) | 상품명 |
| price | int | 가격 |
| event_type | varchar(10) | 1+1 또는 2+1 |
| image_url | varchar(500) | 상품 이미지 URL |

### 데이터 추가
- 기존 `mart` 테이블의 1개 row(id=1, 좌표/주소 null)를 실제 CU 서울시청점 데이터로 업데이트
- CU 광화문점, 명동점, 을지로점, 종각점 추가 등록 (총 5개 매장)
- 80개 상품을 5개 매장에 분배 (`UPDATE product SET mart_id = ...`)

## 백엔드 (Spring Boot)

### 신규 파일

| 파일 | 설명 |
|---|---|
| `entity/Mart.java` | `mart` 테이블 JPA 엔티티 (Lombok `@Getter`, `@Entity`, `@Table`) |
| `repository/MartRepository.java` | Haversine 공식 기반 주변 CU 검색 native query |
| `service/MartService.java` | 주변 CU 검색 비즈니스 로직 |
| `controller/MartController.java` | `GET /api/marts/nearby` 엔드포인트 |

### 수정 파일

| 파일 | 변경 내용 |
|---|---|
| `repository/ProductRepository.java` | `findByMartId(Long martId)` 메서드 추가 |
| `service/ProductService.java` | `getProductsByMartId(Long martId)` 메서드 추가 |
| `controller/ProductController.java` | `GET /api/products?martId=` 쿼리 파라미터 지원 |

### API 엔드포인트

| Method | Path | 파라미터 | 설명 |
|--------|------|---------|------|
| GET | `/api/marts/nearby` | `lat`, `lng`, `radius`(default=1km) | 주변 CU 매장 검색 (거리순 정렬) |
| GET | `/api/products` | `martId` (optional) | 전체 또는 특정 매장의 상품 조회 |

### Haversine 공식 (MySQL Native Query)
```sql
SELECT m.*,
    (6371 * acos(cos(radians(:lat)) * cos(radians(m.latitude))
    * cos(radians(m.longitude) - radians(:lng))
    + sin(radians(:lat)) * sin(radians(m.latitude)))) AS distance
FROM mart m
WHERE m.name = 'CU'
HAVING distance <= :radius
ORDER BY distance
```

## 프론트엔드 (React + KakaoMap)

### 변경 파일
- `frontend/src/App.tsx` — 전체 재작성

### 추가된 기능
1. **주소/장소 검색 입력창** — 상단에 검색바 추가, Enter/버튼으로 검색
2. **KakaoMap Places 키워드 검색 연동** — `libraries=services` 옵션으로 SDK 로드
3. **주변 CU 매장 마커 표시** — 검색 결과 위치 기준 반경 1km 내 CU 매장을 지도 마커로 표시
4. **마커 인터랙션** — 호버 시 InfoWindow로 주소 표시, 클릭 시 해당 매장 행사상품 로딩
5. **매장별 상품 조회** — 선택된 CU 매장의 상품만 우측 사이드바에 표시 (1+1/2+1 필터 유지)

### 데이터 흐름
```
검색창 입력 → KakaoMap Places.search() → lat/lng 획득
    → GET /api/marts/nearby?lat=&lng=&radius=1 → CU 매장 목록
    → 지도 마커 표시
    → 첫 번째 CU 자동 선택
    → GET /api/products?martId= → 상품 목록 표시
```
